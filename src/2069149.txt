Экстракция (извлечение) – называется процесс, проводимый с целью извлечения растворенного вещества с помощью растворителя для экстракции.
					
					Экстрогирование – называется процесс, проводимый с целью разделения жидкости на компоненты с помощью растворителя для экстрагирования.
					Требования к растворителю для экстракции:
 - не должен растворятся в том растворителе, в котором содержится извлекаемое вещество;
 - существенно лучше растворять извлекаемое вещество, чем тот растворитель, из которого оно извлечено;
 - не должен химически взаимодействовать с извлекаемым веществом, ни с растворителем в котором оно находится;
 - должно легко удаляться при выделении извлекаемого вещества;
 - безопасным и нетоксичным.
   Экстракт
   Для извлечения нужного компонента из твердых веществ, достаточно размешать эту смесь с необходимым растворителем, при определенных условиях, отфильтровать полученный размер и затем выделить искомое вещество. Такой процесс проводимый при комнатной температуре называется мацирацией.

 Основное достоинство: низкая рабочая температура процесса, позволяющая извлекать ряд веществ, которые разлагаются при высоких температурах.(Антибиотики). Во многих случаях экстракция применяется в сочетании с ректификацией.
 Переход извлекаемого вещества из исходного раствора в экстрогент происходит до установления равновесия, которая количественно характеризуется коэффициентом распределения: это отношения равновесных концентраций извлекаемого вещества в экстракте и рафинате. Этот коэффициент линейно зависит от концентрации извлекаемого вещества.
 Процесс выщелачивая – это частный случай процесса экстракции. Это извлечение из твердого исходного сырья одного или нескольких веществ при помощи растворителя обладающего избирательной способностью. Вода (как растворитель для экстракции), применяется для извлечения сахара из свеклы, спирт и спиртовые смеси для получения настоек, наливок и т.д. Масло из семян подсолнечника извлекается при помощи бензина. За выщелачиванием в технологической схеме действуют стадии: разделения (фильтрация, выпаривание, кристаллизация), очистки.
	
 Из всех стадий процесса минимизирующей будет, та, которая определяет скорость процесса на первых стадиях. Движущей силой является разность концентраций экстрагируемого вещества на поверхности и объёме.
