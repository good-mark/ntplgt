История Windows Phone: от Windows Mobile до наших дней



Автор Артур Лоянич Windows 10, мобильная версия и новшества

Наиболее актуальная новость на текущий момент в экосистеме Windows Phone – ожидаемый в следующем году выход операционной системы Windows 10.
Не зря это говорится в единственном числе: данная ОС будет единой для всех гаджетов, от смартфона до десктопа.
То есть нынешнее разделение Windows на Windows и Phone уйдет в прошлое. 



Формально новая ОС будет называться Windows 10 Mobile (пока), и ее приоритет в плане синхронизации – расширенный набор синхронизируемого контента и функций, универсальные приложения (то есть для мобильной и настольной версии), возможность подключения гаджетов к внешнему дисплею, использование смартфона в качестве ПК с поддержкой мыши и клавиатуры.
Минимальные требования для системы аналогичны таковым для Windows Phone 8, включая разрешение экрана, оперативную память и частоту процессора.
Будет добавлена и поддержка для новых чипсетов Qualcomm Snapdragon, включая 64-битные.



Что касается интерфейса, то уже известно (а большинство уже показано в Preview-версиях), что будет добавлена поддержка фоновых изображений под плитками, возможность растягивания плиток по вертикали, новый Office Mobile, новый дизайн контекстных меню, полностью синхронизированный с десктопной версией Центр Уведомлений, новый Проводник для навигации по файлам, новое приложение по умолчанию для камеры; калькулятор, совмещенный с приложением по конвертированию одних единиц в другие, LED-уведомления для приложений, увеличенный порог для включения режима экономии заряда, и так далее, и тому подобное.
Потому что проще сказать, что что-то не обновилось, поскольку обновится практически всё: будут либо новые приложения, либо существенное обновление нынешнего.



Первое техническое Preview новой ОС вышло 12 февраля текущего года, оно доступно для некоторых гаджетов Nokia/Microsoft Lumia и HTC One M8.
Новость для обладателей смартфонов на Windows Phone 8

К счастью, устройства на Windows Phone 8.1 действительно будут иметь возможность обновления до «десятки», и это официальная позиция компании Microsoft.
Единственное «но» - некоторые функции устройств могут различаться в зависимости от гаджета.
Но то, что обновление выйдет (нет, все не будет «как с Windows Phone 7») - это уже известный факт.
В пользу которого говорит и то, что на некоторых устройствах (см. выше) обновление до Windows 10 Mobile уже возможно и новые «фишки» на них можно уже попробовать.
Заключение

Формально история Windows Phone на этом не кончается.
Да, эта операционная система обретет новое имя с выходом Windows 10 Mobile, но фактически это все же будет «старая добрая» Windows Phone, до которой смогут обновиться все смартфоны с текущей Windows Phone 8.1 вне зависимости от того, установлен у вас Update 2 или нет – в данном случае он вторичен во всех смыслах.

Поэтому стоит дождаться выхода новой ОС, которая откроет новую главу в истории мобильных операционных систем от Microsoft.
