1.1. По цвету портландцемент подразделяют на: красный, желтый, зеленый, голубой, розовый, коричневый и черный.
 2. 1.2. По механической прочности портландцемент подразделяют на марки: 300, 400 и 500.
 3. 1.3. Портландцемент должен изготовляться в соответствии с требованиями настоящего стандарта по технологическим регламентам, утвержденным в порядке, установленном министерством-изготовителем.
 4. 1.4. Портландцемент должен содержать не менее 80 % клинкера, не более 6 % активной минеральной добавки, не более 15 % минерального, искусственного или природного пигмента или не более 0,5 % органического пигмента от массы цемента.
 5. 1.4.1. Белый клинкер должен быть белизной не менее 68 % абсолютной шкалы по ГОСТ 965-78.
 6. 1.4.2. Допускается для портландцемента желто-красной гаммы и коричневого цвета применять отбеленный клинкер белизной не менее 40 % абсолютной шкалы, а для черного - обыкновенный клинкер.
 7. 1.4.3. Активные минеральные добавки осадочного происхождения белизной не менее 68 % абсолютной шкалы должны удовлетворять требованиям ОСТ 21-9-74.
 8. Допускается для портландцемента желто-красной гаммы и коричневого и черного цветов применять добавки белизной не менее 40 % абсолютной шкалы.
 9. 1.4.4. Красящие пигменты должны обладать щелоче- и светостойкостью, не должны содержать примесей, оказывающих вредное влияние на морозостойкость и прочность цементного камня, и соответствовать нормативно-технической документации на пигменты и красители.
 10. 1.4.5. Гипсовый камень должен удовлетворять требованиям ГОСТ 4013-74.
 11. 1.5. По согласованию с потребителем допускается введение в портландцемент при его помоле поверхностно-активных пластифицирующих и гидрофобизирующих добавок в количестве не более 0,3 % его массы в пересчете на сухое вещество.
 12. 1.6. Допускается вводить в портландцемент специальные добавки, улучшающие его декоративные свойства, в количестве не более 2 % массы цемента.
 13. 1.7. Содержание окиси магния (MgO) в клинкере не должно быть более 5 %, содержание свободной окиси кальция (СаОсв.) не должно быть более 1,5 % по массе.
 14. 1.8. Содержание ангидрида серной кислоты (SO3) в портландцементе не должно быть более 3,5 % массы цемента.
	
 15. 1.9. Портландцемент должен быть однородным по цвету в пределах отгружаемой партии и сохранять свой цвет при тепловлажностной обработке и воздействии ультрафиолетовых лучей.
 16. 1.10. Цвет портландцемента должен соответствовать эталону, утвержденному Минстройматериалов СССР.
 17. 1.11. Эталоном служит образец портландцемента или цементная окраска. Образцы-эталоны утверждаются для каждого завода.
 18. 1.12. Предел прочности образцов из цемента, изготовляемых и испытанных по ГОСТ 310.4-81 через 28 сут с момента изготовления, должен быть не менее значений, указанных в таблице.

  19. 1.13. Начало схватывания цемента должно наступать не ранее 45 мин, а конец - не позднее 12 ч от начала затворения.
 20. 1.14. Портландцемент должен показывать равномерность изменения объема при испытании образцов кипячением в воде.
 21. 1.15. Тонкость помола цемента должна быть такой, чтобы при просеивании пробы сквозь сито с сеткой № 008 по ГОСТ 6613-86, проходило не менее 90 % массы просеиваемой пробы.
 22. 1.16. Цемент высшей категории качества должен удовлетворять следующим дополнительным требованиям:
 23. обладать стабильными показателями прочности при сжатии, коэффициент вариации прочности для цемента марок 300 и 400 должен быть не более 5 %, а для цемента марок 500 - не более 3 %.
 24. Красящие пигменты должны удовлетворять следующим нормативно-техническим документам:
 25. для цементов красного и розового цвета - руда железная красковая гематитовая по ТУ 14-9-71-74;
 26. для цементов голубого цвета - голубой фталоцианиновый пигмент по ГОСТ 6220-76;
 27. для пигментов желтого цвета - желтый железоокисный пигмент по ГОСТ 18172-80;
 28. для цементов зеленого цвета - зеленый фталоцианиновый пигмент по ТУ 6-14-488-76;
 29. для цементов коричневого цвета - смесь руды железной красковой гематитовой по ТУ 14-9-71-74 и пероксида по ТУ 14-9-50-73
