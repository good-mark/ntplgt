1. При вводе данных в список можно заполнять ячейки точно так же, как это делается при заполнении обычного листа. Если в списке много повторяющихся значений, то при его формировании окажет помощь автоввод.
 2. Другой способ ввода данных – использование формы. Форма – способ отображения данных, когда на экране располагаются поля только одной записи-строки.
 3. Создать строку заголовка и одну запись.
 4. Поместить курсор в любую ячейку списка и выбрать команду Данные®Форма.
 5. В диалоговом окне, используя полосу прокрутки перейти на вторую запись, ввести новую запись. Для перехода от одного поля ввода к другому использовать клавишу Tab.
 6. Закончив ввод информации в поля, щелкнуть на кнопке Добавить.
 7. Ввести следующую запись, щелкнуть на кнопке Добавить. Закончив ввод данных по всем записям, щелкнуть на кнопке Закрыть.
					Поиск данных в списках
					1. Для поиска информации с помощью заданных критериев поиска используется форма.
 2. Выбрать команду Данные®Форма.
 3. Щелкнуть на кнопке Критерии, форма превращается в инструмент поиска.
 4. Ввести требуемые критерии поиска в подходящие поля.
 5. Нажать клавишу Enter или кнопку Правка, чтобы начать поиск. С помощью кнопок Назад и Далее можно просмотреть все записи в таблице, удовлетворяющие заданному набору критериев.
 Например, при поиске всех фамилий, начинающихся с буквы П, в соответствующем поле указывается одна буква. Если необходимо найти записи о сотрудниках 1 отдела, поступивших на работу после 01.06.96, то в поле Отдел указывается номер 1, а в поле Дата поступления условие >01.06.96.
					Фильтрация данных в списке
					Фильтры являются инструментом поиска для списков в Excel. Фильтрация позволяет находить и отбирать для обработки часть записей (строк), которые отвечают определенным критериям. Остальные строки при этом скрыты.
 Для отбора данных списка используется Автофильтр и Расширенный фильтр.
 Для установки Автофильтра следует:
 1) находясь в пределах списка, выбрать команду Данные®Фильтр®Автофильтр;
 2) заголовки столбцов преобразуются в раскрывающиеся списки, в которых можно задавать нужные критерии для поиска данных. В раскрывающемся списке выводятся все значения, встречающиеся в столбце, и дополнительные опции: Все, Первые 10, Условие, Пустые или Непустые.
